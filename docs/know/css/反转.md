利用CSS的`filter`属性做黑夜白昼模式
<!-- more -->
***

## **filter**
``` css
filter: invert(100%) contrast(87%);
```
在html上增加此样式即可以做到亮色主题反转为黑夜模式

::: waring 缺点
图片和视频也会被翻转颜色
:::
